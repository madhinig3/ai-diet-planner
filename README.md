\# AI-Based Personalized Diet Planner



\## Project Description

This project extracts medical information from patient reports using OCR and NLP

and generates a personalized diet recommendation using AI.



\## Technologies Used

\- Python

\- Streamlit

\- Tesseract OCR

\- OpenCV

\- HuggingFace BERT

\- GitHub

\- Streamlit Cloud



\## Features

\- Upload medical report (PDF/Image)

\- OCR-based text extraction

\- Patient name and doctor's summary extraction

\- AI-based diet recommendation



\## How to Run

1\. Upload a medical report

2\. View extracted patient data

3\. Get personalized diet suggestion
